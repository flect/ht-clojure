ht-clojure
==================

Sample project that use Clojure for Heroku training.

How to deploy
--------------

### Step1  

Clone app from Github  

```
$ git clone git@github.com:flect/ht-clojure.git
# cd ht-clojure/
```

### Step2  

Create Heroku application

```
$ heroku create
```

### Step3  

Deploy application

```
$ git push heroku master
```

### Step 4

Visit application

```
$ heroku open
```
